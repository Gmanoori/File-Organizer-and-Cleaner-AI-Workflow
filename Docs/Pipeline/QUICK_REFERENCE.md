## 📋 Implementation Phases

### **Phase 1: Setup (Days 1-2, ~3 hours)**

**What to do:**
```bash
# 1. Install Docker & Docker Compose
curl -o docker-compose.yml \
  https://raw.githubusercontent.com/kestra-io/kestra/develop/docker-compose.yml
docker compose up -d

# 2. Install Spark
brew install apache-spark  # or apt install on Linux

# 3. Install Python libraries
pip3 install pandas pymongo anthropic pyspark

# 4. Create project directory
mkdir -p ~/kestra-project/{flows,scripts,docs}
cd ~/kestra-project

# 5. Get API keys
# - MongoDB Atlas: https://www.mongodb.com/cloud/atlas
# - Claude API: https://console.anthropic.com

# 6. Create .env file
cat > .env << 'EOF'
MONGO_CONNECTION_STRING="mongodb+srv://user:pass@cluster0.mongodb.net/data_cleaning"
ANTHROPIC_API_KEY="sk-ant-..."
DATA_PATH="/tmp/organized_data"
EOF
```

**Verify setup:**
```bash
kestra --version           # Should work after Docker up
spark-submit --version    # Should show Spark version
python3 -c "import pymongo; print('✓ pymongo')"
python3 -c "import anthropic; print('✓ anthropic')"
```

## 🐛 Troubleshooting Quick Reference

### **Setup Issues**

| Error | Cause | Fix |
|-------|-------|-----|
| `docker: command not found` | Docker not installed | Install Docker Desktop |
| `spark-submit: command not found` | Spark not in PATH | Add to ~/.bashrc: `export PATH=$PATH:/usr/local/spark/bin` |
| `ModuleNotFoundError: pandas` | pip library not installed | `pip install pandas` |

### **Kestra UI Issues**

| Error | Cause | Fix |
|-------|-------|-----|
| Can't access http://localhost:8080 | Docker container not running | `docker compose up -d` |
| "Secret not found" | Secret name wrong | Check `{{ secret('exact_name') }}` matches |
| Task stuck/hanging | Infinite loop or timeout | Add `timeout: PT5M` to task config |

### **Data Issues**

| Error | Cause | Fix |
|-------|-------|-----|
| "File not found" | Path uses `~` in Kestra | Use `/home/user/` instead of `~/` |
| MongoDB connection error | IP not whitelisted | Go to MongoDB Atlas → Network Access → Add 0.0.0.0/0 |
| Claude API rate limit (429) | Too many calls at once | Reduce batch size, add retry logic |
